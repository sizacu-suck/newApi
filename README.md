# newApi
API для управления каталогом товаров и категорий интернет-магазина. Проект разработан на платформе .NET с использованием современных подходов к проектированию серверной части (DTO, LINQ-проекции, валидация данных).

Категории
GET /api/categories — Получить список всех категорий.
GET /api/categories/{id} — Получить категорию по уникальному ID.
POST /api/categories — Создать новую категорию (с валидацией входных данных).
PUT /api/categories — Изменить название существующей категории.
DELETE /api/categories/{id} — Удалить категорию из базы данных.

Товары
GET /api/products — Получить список всех товаров.
GET /api/products/{id} — Получить конкретный товар по ID. Возвращает оптимизированный DTO с текстовым названием категории.
POST /api/products — Добавить новый товар с привязкой к существующей категории (Foreign Key).
PUT /api/products — Обновить информацию о товаре.
DELETE /api/products/{id} — Удалить товар из системы.
